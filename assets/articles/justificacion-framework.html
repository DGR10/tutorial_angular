Justificación del Framework Angular

1. Justificación del Framework:

  • a. ¿Para qué sirve?

    Angular es un framework de desarrollo frontend utilizado para crear aplicaciones web dinámicas y modernas de una sola página (SPA, por sus siglas en inglés).

  • b. ¿Qué aplicaciones tiene?

    Angular es ideal para:

    • Aplicaciones web SPA.
    • Sistemas de gestión empresarial (ERPs, CRMs).
    • Aplicaciones móviles híbridas (con Ionic).
    • Portales web interactivos.

  • c. ¿Sobre qué lenguajes se apoya?

    Angular se basa principalmente en:

    • TypeScript: Un superconjunto de JavaScript que añade tipado estático.
    • HTML: Para el diseño de la estructura visual.
    • CSS: Para los estilos y diseño responsivo.

  • d. ¿Qué prerrequisitos necesita?

    Para trabajar con Angular se necesitan:

    • Conocimientos básicos de HTML, CSS y JavaScript.
    • Familiaridad con TypeScript (recomendado).
    • Node.js instalado para gestionar dependencias.
    • Angular CLI para facilitar la creación y administración de proyectos.
