Primeros Pasos con Angular

3. Primeros Pasos:

a. Cómo realizar una primera aproximación ("Hola Mundo")

  1. Instalar Angular CLI: Asegúrate de tener Angular CLI instalado ejecutando en la terminal: 
    npm install -g @angular/cli
  2. Crear un nuevo proyecto Angular: Ejecuta el siguiente comando y sigue las instrucciones para configurar el proyecto: 
    ng new hola-mundo
  3. Navegar al directorio del proyecto: 
    cd hola-mundo
  4. Iniciar el servidor de desarrollo: Usa el siguiente comando para iniciar la aplicación: 
    ng serve
    Accede a la aplicación en tu navegador en la URL: http://localhost:4200
  5. Modificar el componente principal: Edita el archivo src/app/app.component.html y reemplázalo con el siguiente contenido: 
    <h1>¡Hola Mundo desde Angular!</h1>
  6. Guardar y visualizar: Guarda los cambios. La aplicación se recargará automáticamente mostrando el mensaje "¡Hola Mundo desde Angular!" en tu navegador.
