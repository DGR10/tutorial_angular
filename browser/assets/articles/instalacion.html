Guía de Instalación de Angular

2. Instalación:

  • a. ¿Se precisa instalación?

    Sí, Angular requiere la instalación de algunas herramientas como Node.js y Angular CLI.

  • b. ¿Qué pasos hay que realizar para realizar la instalación?

    1. Descargar e instalar Node.js (versión LTS recomendada).
    2. Instalar Angular CLI globalmente ejecutando: npm install -g @angular/cli
    3. Crear un nuevo proyecto Angular con: ng new nombre-del-proyecto

  • c. ¿Versiones necesarias?

    Se recomienda usar:

    • Node.js: Versión 16.x o superior.
    • Angular CLI: Última versión estable disponible.

  • d. ¿Rutas para tener en cuenta?

    El directorio del proyecto debe ser accesible y los comandos se ejecutan en el terminal desde esa ubicación.

  • e. ¿Variables de entorno?

    No es necesario configurar variables de entorno para una instalación básica, pero para configuraciones avanzadas, puede ser necesario definir NODE_ENV.

  • f. ¿Necesita base de datos?

    No de manera predeterminada. Angular se enfoca en el frontend y consume APIs externas, que podrían conectarse a bases de datos.

  • g. Otros aspectos a considerar:

    Verificar que npm esté actualizado ejecutando npm install -g npm.
